>> Course Selling App (No React)

A full-stack course selling platform built using Node.js, Express, MongoDB, and vanilla JS.

>> Features
- User signup & login (JWT auth)
- Admin course creation
- Purchase courses
- Dynamic frontend (no framework)

>> Tech Stack
- Backend: Node.js, Express
- Database: MongoDB (Mongoose)
- Auth: JWT, bcrypt
- Frontend: HTML, CSS, Vanilla JS

>> Setup:
npm install
node index.js
